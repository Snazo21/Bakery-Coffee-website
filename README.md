# Bakery-Coffee-website
Level1 - Task 1 - Codedev internship

A static website of basic HTML and CSS, designed to display goods, coffee and refreshing drinks made by artisans. It focuses on designing for the user and user experience, not the teachnology itself. Tight coding standards were maintained.

## Technology stack

* **HTML5:** Semantic tags for the structure for easy reach and Search Engine Optimization(SEO).
* **CSS:** Styling and warmth and ensuring a full responsive layout, including the menu button, dark and light mode and screen size responsiveness

## Challenges

Researching on which tags are suitable for a certain part of the website, e.g the svg tag for the animated barcode, article tag for the card section and section for the form

## Lessons

Inclusive design for the people with fully responsive layouts and accessibility
